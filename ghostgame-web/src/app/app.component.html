Optimal Ghost Game! Instructions In the game of Ghost, two players take turns building up an English word from left to right. Each player adds one letter per turn. The goal is to not complete the spelling of a word: if you add a letter that completes a word (of 4+ letters), or if you add a letter that produces a string that cannot be extended into a word, you lose. Basically, each player should try to extend the game as much as possible so the rival loses if one of those conditions are met
